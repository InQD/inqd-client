## 🎄 프로젝트 트리

```
src
ㄴassets     // 이미지, json 등
ㄴcomponents // 전역 컴포넌트
ㄴhooks
ㄴpages      // 페이지
ㄴroutes     // 라우터
ㄴservices
ㄴstates     // 전역 상태관리
ㄴstyles     // 전역 스타일
ㄴtypes      // 전역 타입
ㄴutils      // 전역 함수
```
